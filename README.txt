UPLOAD TO GITHUB:
1. Extract this ZIP.
2. Open your GirlsTrip repository on GitHub.
3. Choose Add file > Upload files.
4. Drag every extracted file into the upload area.
5. Allow GitHub to replace files with the same names.
6. Commit changes.
7. Wait 1–3 minutes.
8. Open the live site and press Ctrl+F5.

IMPORTANT:
The site is static. Kristi's selections will not automatically sync with yours.
Use Planner > Download Trip Plan to send the plan file between devices.
