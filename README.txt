UPLOAD:
1. Extract this ZIP.
2. Open the GitHub GirlsTrip repository.
3. Choose Add file > Upload files.
4. Upload every extracted file and replace files with the same names.
5. Commit the changes.
6. Wait 1–3 minutes.
7. Open the live site and press Ctrl+F5.
